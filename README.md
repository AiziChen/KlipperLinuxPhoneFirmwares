# KlipperLinuxPhoneFirmwares

Yet Another Klipper Firmwares for some Linux Phones

Now supported smart phones list:

* Redmi Note4x
* Xiaomi Max 2
* Xiaomi Note LTE
* ZUK Z1
* Meizu M6 Note (later)


# Features

Currently the firmware uses Ubuntu 24.04, Linux kernel 7.0.1

* Best performance
* 


# How to flash

make sure your phone is unlocke!!!

* download firmware of your phone
* turn your phone into `BOOTLOADER` mode
* run flash script under Terminal application

for Unix(Linux/macOS):
> sh flash-all.sh

for Windows:
> .\flash-all.bat


# Enjoy

If you get luck, your will turn on to the klipperScreen and enjoy printing! If your phone turn on to black screen, please create an issue about it.
